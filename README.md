This repository contains data and R code used to calculate rates and plots of uterine cancer in Australian women from 2001-2020. 
When you have all of the excel files in the main branch saved in your working directory and run the code in the R script files, this will produce the calculation of rates and plot results from joinpoint analyses.
Switch to the PAFs branch to obtain excel files for calculations of PAFs.
Switch to the Joinpoint analyses branch to find joinpoint files with results for each joinpoint analysis.
